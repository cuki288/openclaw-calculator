### Problem
작업 중 중간에 답장이 끊기고 시간 초과 (timeout) 문제가 발생.

1. **Gateway Socket Timeout:** 10s 타임아웃이 긴 응답 처리에 부적합

3. **Streaming Disabled:** 전체 응답 대기 후 전송으로 중간 끊김 발생
4. **Memory Compaction:** 메모리 컴팩트/플래시 과정에서 작업 인터럽트

- [ ] Gateway timeout 증가 (10s → 120s) ✅ 완료

- [ ] 컨텍스트 토큰 제한 조정 (200k → 150k) ✅ 완료
